# TECHORA — Shopify Theme

## Quick Upload Instructions

1. Zip this entire folder → `techora-theme.zip`
2. Go to Shopify Admin → Online Store → Themes
3. Click "Add theme" → "Upload zip file"
4. Upload `techora-theme.zip`
5. Click "Customize" to start editing

## PayPal Setup

1. Go to https://developer.paypal.com
2. Create an app → copy the **Live Client ID**
3. In Shopify Admin → Settings → Payments → PayPal
4. Connect your PayPal Business account (payments@youremail.com)

## Setting Up Navigation

1. Shopify Admin → Online Store → Navigation
2. Create a menu called "Main menu" with links to your collections
3. Create a menu called "Footer Shop" for footer links

## Recommended Collections to Create

- /collections/wearables
- /collections/accessories

## Customization
All sections are editable via Shopify Theme Editor:
- Hero text, buttons, badge
- Categories (add/remove/edit each card)
- Featured Products (pick any collection)
- Payment section text
- Newsletter text
- Footer links & social URLs

## Dropshipping Notes (India → US)
- Set shipping zones to USA only in: Settings → Shipping
- Use AliExpress/CJ Dropshipping + DSers app for product sourcing
- Enable PayPal as payment method (you'll receive USD)
- Use Shopify's built-in currency conversion if needed
